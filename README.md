## SEO Features

- Schema.org JSON-LD structured data (Person, LegalService, FAQPage, BreadcrumbList, ItemList, Organization)
- Semantic HTML5 with proper heading hierarchy
- Open Graph and Twitter Card meta tags
- XML sitemap and robots.txt
- Mobile-first responsive design
- Optimized title tags and meta descriptions per page
- Internal linking between profiles, practice areas, and content pages
- FAQ accordion with FAQPage schema for rich snippets
- Legal disclaimers (attorney advertising, no legal advice)
- WCAG 2.1 AA accessibility features (skip links, ARIA labels, keyboard navigation, color contrast)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, media queries
- **Vanilla JavaScript** — No framework dependencies
- **No build step required** — Deploy as static files
